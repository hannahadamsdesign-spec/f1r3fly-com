Code Accuracy Guarantees

Delivery of Self Testing Code

F1R3FLY's OSLF (Operational Semantics in Logical Form) is a tool that dynamically checks computer code, validating its program type and checking for faults and vulnerabilities. If the code is found to be safe, OSLF dry-runs it and reports on what its outcome would be. This functionality has been tested using examples of notorious hacks and software failures and has consistently identified the weaknesses that let them through.
